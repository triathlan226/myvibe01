# 🧠 記憶大挑戰 - Memory Match Game

一個有趣且具有挑戰性的記憶翻牌配對遊戲！

## 🎮 遊戲特色

- **三種難度等級**
  - 簡單 (4×3)：6 對卡片
  - 中等 (4×4)：8 對卡片
  - 困難 (6×4)：12 對卡片

- **遊戲功能**
  - 即時計時器
  - 翻牌次數統計
  - 配對進度顯示
  - 3 次提示機會
  - 過關評分系統（⭐⭐⭐）

- **精美設計**
  - 流暢的翻牌動畫
  - 漂亮的漸層背景
  - 響應式設計，支援手機和平板

## 🚀 如何開始

1. 解壓縮 zip 檔案
2. 用瀏覽器開啟 `index.html`
3. 選擇難度等級
4. 開始翻牌配對！

## 🎯 遊戲規則

1. 點擊卡片翻開
2. 每次翻開兩張卡片
3. 如果圖案相同，卡片保持翻開
4. 如果圖案不同，卡片會翻回去
5. 找出所有配對即可過關

## 💡 小技巧

- 記住卡片位置是致勝關鍵
- 善用提示功能找出難找的配對
- 挑戰更高難度獲得更好的成就感！
